# Quickstart

* Configure the following items:
  * `IAR: Select Workbench`
  * `IAR: Select Compiler`
  * `IAR: Select Project`
  * `IAR: Select Configuration`

* Configure an `IAR - build` task using `Tasks: Configure Task`
  
* Build your project
